# CNet - Simple Packet Sniffer in C

A lightweight packet sniffer built using raw sockets (AF_PACKET).
It parses Ethernet, IPv4, TCP and UDP headers.

## Installation

### 1. Clone the repository

```
git clone https://github.com/julioSantos-sys/cnet.git
cd cnet
```

### 2. Run the makefile

```
make install
```
### 3. Run the cnet with root

```
sudo cnet
```
